# 📂 Peer-to-Peer File Transfer System

## 📌 Overview

This project is a Peer-to-Peer (P2P) file transfer application built using Python. It allows multiple users (peers) to connect, search, and download files directly from each other without a centralized server.

## 🚀 Features

* 🔍 Search files across peers
* ⬇ Download files from connected peers
* 🔗 Socket-based communication
* ⚡ Multithreading for handling multiple clients
* 🖥️ GUI built using Tkinter
* 📂 File registration system

## 🛠️ Technologies Used

* Python
* Socket Programming
* Multithreading
* Tkinter (GUI)

## 📁 Project Structure

```
P2P.py          # Main application file
downloads/      # Downloaded files folder
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/p2p-file-transfer.git
```

2. Navigate to project folder:

```
cd p2p-file-transfer
```

3. Run the application:

```
python P2P.py
```
